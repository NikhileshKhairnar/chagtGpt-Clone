# 🤖 ChatGPT Clone

An AI-powered chatbot web application that generates intelligent responses to user queries in real time. The project provides a clean and responsive chat interface, allowing users to interact with an AI model through a simple and intuitive experience.

## 🚀 Features

- 💬 AI-powered chatbot
- ⚡ Real-time response generation
- 📱 Responsive user interface
- 📋 Copy response functionality
- 📝 Chat history
- ⚠️ Error handling for failed API requests
- 🎨 Modern and clean UI

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Gemini API

## 📂 Project Structure

```
ChatGPT-Clone/
│── index.html
│── style.css
│── script.js
│── assets/
│── README.md
```

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/NikhileshKhairnar/chagtGpt-Clone.git
```

2. Open the project folder.

3. Add your Gemini API key in the JavaScript file.

4. Run `index.html` in your browser.
